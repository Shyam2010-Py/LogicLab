3. Karnaugh Map Rules

  • Group only adjacent 1s (or 0s for POS) — groups must be powers of 2 (1, 2, 4, 8, 16)
  • Groups can wrap around edges
  • Make groups as large as possible
  • Every 1 must be covered by at least one group
  • 2 variables → 2×2 map · 3 variables → 2×4 map · 4 variables → 4×4 map
  • n variables → map has 2n cells

Common Groupings

  • Single 1 → minterm (3 variables for 4-cell map)
  • Pair of 1s → eliminates 1 variable (2 variables)
  • Quad → eliminates 2 variables (1 variable)
  • Octet → eliminates 3 variables (constant 1)

8. Quick Exam Shortcuts

  • 2n values: 2¹=2, 2²=4, 2³=8, 2⁴=16, 2⁵=32, 2⁶=64, 2⁷=128, 2⁸=256, 2¹⁰=1024
  • 2's complement range: For n-bit signed number: -2n-1 to +(2n-1 − 1)
  • 2's complement = 1's complement + 1
  • Negative number in 2's comp: Invert all bits and add 1
  • Excess-2n-1 (Bias): Used in floating point. E' = E + 2n-1
  • Gray code: Each successive value differs by only 1 bit
  • Parity bit: Adds bit to make total 1s even (even parity) or odd (odd parity)
  • Hamming distance: Number of bit positions where two codes differ

10. Important Definitions

  • Combinational CircuitOutput depends only on current inputs (no memory).
  • Sequential CircuitOutput depends on current inputs AND previous state.
  • Flip-Flop1-bit memory element; edge-triggered bistable circuit.
  • LatchLevel-sensitive 1-bit memory (transparent when enabled).
  • Propagation DelayTime for a signal change to propagate from input to output.
  • Setup TimeMinimum time data must be stable BEFORE clock edge.
  • Hold TimeMinimum time data must remain stable AFTER clock edge.
  • Race ConditionOutput depends on unpredictable order of signal changes.
  • Fan-OutMaximum number of inputs a gate output can drive.
  • Fan-InNumber of inputs a gate has.
  • Min/MaxtermMinterm = product (AND) of all variables. Maxterm = sum (OR) of all variables.
  • SOP / POSSum of Products / Product of Sums — standard forms.
