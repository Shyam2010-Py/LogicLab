📖 About Complements

Complements simplify subtraction in digital systems. Instead of building dedicated subtractor circuits, computers add the 2's complement of a number to perform subtraction.

1's Complement: Flip every bit (0↔1). Example: 1's complement of 1010 = 0101.

2's Complement: 1's complement + 1. Used to represent signed numbers in binary.

Subtraction using 2's Complement: A − B = A + (2's complement of B). If result overflows, discard the carry-out.

Examples

  • 1's comp(0) = 1, 2's comp(0) = 0
  • 1's comp(1) = 0, 2's comp(1) = 1
  • 1's comp(1010) = 0101, 2's comp(1010) = 0110
  • 1's comp(00001111) = 11110000, 2's comp(00001111) = 11110001
