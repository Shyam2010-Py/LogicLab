🔀 Multiplexer (MUX)

A data selector that routes one of multiple inputs to a single output line.

📖 Definition

A Multiplexer (MUX) is a combinational circuit that selects one of several input data lines and forwards the selected input to a single output line. The selection is controlled by select lines.

An N-to-1 MUX has 2N data inputs and N select lines.

⚙️ General Equation

Y = Σ(mi · Di)

Where mi are the minterms from select lines and Di are the data inputs.

2-to-1 Multiplexer

Selects between 2 data inputs (D₀, D₁) using 1 select line (S).

Circuit Symbol

Boolean Expression

Y = S'·D₀ + S·D₁

Truth Table

Select (S)Output (Y)
0D₀
1D₁

Internal Circuit

   D₀ ──────┐
            ├──[NOT S]─[AND]──┐
   S' ──────┘                  │
                               ├──[OR]──── Y
   D₁ ──────┐                  │
            ├─────[AND]───────┘
   S  ──────┘

4-to-1 Multiplexer

Selects one of 4 data inputs (D₀–D₃) using 2 select lines (S₁, S₀).

Circuit Symbol

Boolean Expression

Y = S₁'S₀'D₀ + S₁'S₀D₁ + S₁S₀'D₂ + S₁S₀D₃

Truth Table

S₁S₀Output (Y)
00D₀
01D₁
10D₂
11D₃

💡 Applications

  • Data Routing: Selecting between multiple data sources
  • Communication Systems: Time-division multiplexing (TDM)
  • Memory Addressing: Selecting memory banks
  • Function Implementation: Any Boolean function can be implemented using MUXes (universal)
  • ALU Design: Selecting between arithmetic and logic outputs
  • Parallel-to-Serial Conversion: Sequential selection of parallel bits

Advantages

  • Reduces number of wires needed
  • Cost-effective for selecting data
  • Can implement any Boolean function of n variables using 2n-1-to-1 MUX
