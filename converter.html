📖 About Number Systems

A number system defines how numbers are represented. Digital systems use binary because electronic circuits easily represent two states (0 and 1).

Positional Notation: Each digit's value depends on its position. For example, in decimal 234, the 2 represents 200 (2 × 10²).

Common Conversions:

  • Decimal → Binary: Repeated division by 2, read remainders bottom-up
  • Binary → Decimal: Multiply each bit by 2^position and sum
  • Binary ↔ Hex: Group binary digits in sets of 4
  • Binary ↔ Octal: Group binary digits in sets of 3
