➗ Half Adder Circuit

The fundamental 1-bit adder with Sum and Carry outputs.

📖 Theory

A Half Adder is a combinational circuit that adds two single binary bits. It produces two outputs:

  • Sum (S): A ⊕ B (XOR)
  • Carry (C): A · B (AND)

It is called "half" because it does not handle a carry input from a previous stage (full adders do).

⚙️ Boolean Expressions

Sum: S = A ⊕ B
Carry: C = A · B

🔧 Circuit Implementation

The half adder uses two logic gates:

  • 1 × XOR gate: Computes the Sum output (S = A ⊕ B)
  • 1 × AND gate: Computes the Carry output (C = A · B)

💡 Examples

  • 0 + 0: Sum = 0, Carry = 0 (no addition needed)
  • 0 + 1: Sum = 1, Carry = 0 (result fits in 1 bit)
  • 1 + 0: Sum = 1, Carry = 0 (same as above)
  • 1 + 1: Sum = 0, Carry = 1 (result is 10 in binary)

Limitations

A half adder cannot add a carry-in bit from a previous stage. For multi-bit addition, use a Full Adder instead, which accepts three inputs (A, B, Cin).
