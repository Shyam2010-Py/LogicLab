💡 Key Concepts

  • Universal Gates: NAND and NOR can implement ANY logic function alone — this is the foundation of modern digital ICs.
  • Active Levels: NAND/NOR outputs are "active low" (true when 0). Engineers use bubbles to indicate inversion.
  • XOR Properties: A⊕A = 0, A⊕A' = 1, A⊕0 = A, A⊕1 = A'. Useful for controlled inverters and adders.
  • IC Packages: 7400 (NAND), 7402 (NOR), 7408 (AND), 7432 (OR), 7486 (XOR) — common TTL gates.
