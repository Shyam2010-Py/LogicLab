🔁 Flip-Flops

Sequential memory elements that store 1 bit of data — SR, JK, D, and T flip-flops.

💡 Key Concept: A flip-flop is a bistable circuit that stores one bit. Unlike combinational logic, its output depends on both current inputs AND previous state (memory).

The notation Qt refers to the current state, and Qt+1 refers to the next state (after clock edge).

SR (Set-Reset) Flip-Flop

Definition: The most basic flip-flop with two inputs — Set (S) and Reset (R). Setting S=1 stores 1; resetting R=1 stores 0. Both S=R=1 is invalid/forbidden.

Symbol

Truth Table

ClockSRQt+1Action
↑00QtNo change
↑010Reset
↑101Set
↑11XForbidden / Invalid

Characteristic Equation

Qt+1 = S + R'·Qt   (with S·R = 0)

Advantages

  • Simplest flip-flop — easy to understand and build
  • Direct set/reset capability
  • Used as a basic 1-bit memory cell

Disadvantages

  • Invalid state when S = R = 1
  • Race-around condition in level-triggered SR latch

Applications

  • Basic memory storage in registers
  • Switch debouncing circuits
  • Control logic and latching circuits

JK Flip-Flop

Definition: An improved version of SR. When J=K=1, instead of invalid state, the output toggles (flips to its complement).

Symbol

Truth Table

ClockJKQt+1Action
↑00QtNo change
↑010Reset
↑101Set
↑11Q'tToggle

Characteristic Equation

Qt+1 = J·Q't + K'·Qt

Advantages

  • No invalid state — toggles when J=K=1
  • Versatile — can function as SR, D, or T flip-flop
  • Race-around condition can be eliminated with master-slave or edge-triggered design

Disadvantages

  • More complex internal circuitry than SR
  • Requires careful clock timing

Applications

  • Counters (binary, ripple, synchronous)
  • Shift registers
  • Frequency dividers
  • Memory registers

D (Data / Delay) Flip-Flop

Definition: Has a single data input (D). On clock edge, the output takes the value of D — it's a 1-bit memory that captures the data.

Symbol

Truth Table

ClockDQt+1
↑00
↑11

Characteristic Equation

Qt+1 = D

Advantages

  • Single input — simpler than JK
  • No invalid states
  • Perfect for data storage / registers

Disadvantages

  • Cannot toggle without external logic
  • Doesn't support all flip-flop functions natively

Applications

  • Shift registers
  • Data storage registers
  • Memory cells in RAM
  • Input synchronization in digital systems
  • Pipelining in processors

T (Toggle) Flip-Flop

Definition: Single-input flip-flop. When T=1, output toggles on each clock edge; when T=0, output remains unchanged.

Symbol

Truth Table

ClockTQt+1Action
↑0QtNo change
↑1Q'tToggle

Characteristic Equation

Qt+1 = T ⊕ Qt = T·Q't + T'·Qt

Advantages

  • Simple toggle behavior — perfect for counters
  • Single input simplifies design
  • Can be derived from JK by tying J = K = T

Disadvantages

  • Limited functionality (only hold or toggle)
  • Cannot directly store arbitrary values without external control

Applications

  • Binary counters (each T flip-flop divides frequency by 2)
  • Frequency dividers
  • Toggle switches in control circuits
  • Modulo-N counters (with reset logic)
