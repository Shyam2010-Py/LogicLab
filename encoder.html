📤 Encoder

Converts 2N input lines into N output lines (binary code for the active input).

📖 Definition

An Encoder is a combinational circuit that converts an active input signal into a coded output (usually binary). For 2N inputs, it produces N outputs.

Two common types:

  • Standard Encoder: Only one input can be active at a time. If multiple are active, output is undefined.
  • Priority Encoder: Handles multiple simultaneous inputs by giving priority to the highest-order input.

4-to-2 Binary Encoder

Encodes 4 input lines into 2-bit binary output. Only one input is assumed active at a time.

Boolean Expressions

Y₁ = D₂ + D₃
Y₀ = D₁ + D₃

Truth Table

D₀D₁D₂D₃Y₁Y₀
100000
010001
001010
000111

⚠️ All inputs = 0 produces Y₁Y₀ = 00, which is ambiguous (same as D₀ active).

8-to-3 Priority Encoder

Encodes 8 inputs into 3-bit binary. D₇ has highest priority; the highest active input determines the output.

Boolean Expressions

Y₂ = D₄ + D₅ + D₆ + D₇
Y₁ = D₂·D₃' + D₃·D₂'·D₁' + D₆·D₅' + D₇·D₆'
Y₀ = D₁·D₃'·D₅'·D₇' + D₃·D₅'·D₇' + D₅·D₇' + D₇

Truth Table

D₇D₆D₅D₄D₃D₂D₁D₀Y₂Y₁Y₀
00000001000
0000001x001
000001xx010
00001xxx011
0001xxxx100
001xxxxx101
01xxxxxx110
1xxxxxxx111

x = don't care (lower priority inputs are ignored)

💡 Applications

  • Keyboard Encoding: Each key produces a unique code
  • Interrupt Controllers: Prioritizing multiple interrupt requests
  • Address Encoding: Memory chip selection
  • Data Compression: Reducing N-bit data to log₂(N) bits when only one is active
  • Priority Arbitration: CPU bus arbitration

Common ICs

  • 74148: 8-to-3 Priority Encoder
  • 74147: 10-to-4 Priority Encoder (BCD)
  • 74348: 8-to-3 Priority Encoder with tri-state outputs
