Current

🎉 Initial Release

The very first public release of LogicLab — a complete Digital Electronics learning platform.

📚 Features Added

  • Number System Converter — Binary, Octal, Decimal, Hex conversions with step-by-step explanations
  • Logic Gates Simulator — Interactive AND, OR, NOT, NAND, NOR, XOR, XNOR gates with SVG circuit visualization
  • Truth Table Generator — Generate and export truth tables as CSV
  • Binary Arithmetic — Addition, Subtraction, Multiplication, Division with carry/borrow visualization
  • Complement Calculator — 1's and 2's complement with detailed steps
  • Half Adder — Interactive simulator with truth table and circuit diagram
  • Full Adder — Interactive simulator with full implementation details
  • Flip-Flops — SR, JK, D, T flip-flops with live state log
  • Multiplexer — 2:1 and 4:1 MUX simulators
  • Demultiplexer — 1:2 and 1:4 DEMUX simulators
  • Encoder — 4-to-2 standard and 8-to-3 priority encoder
  • Decoder — 2-to-4, 3-to-8, and BCD-to-7-segment decoders
  • Formula Sheet — Comprehensive formula reference
  • Notes — Concise study notes with search
  • Quiz System — 110+ questions across 8 categories
  • About Page — Project info and learning objectives
  • Changelog — You're looking at it!

⚙️ Platform Features

  • Dark/Light theme toggle (saved to localStorage)
  • Global search across all topics
  • Mobile-first responsive design
  • Bottom navigation on mobile

  • Toast notifications
  • Keyboard accessibility

  • Conversion history (saved to localStorage)

  • Performance statistics
