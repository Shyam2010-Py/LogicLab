📥 Decoder

Converts binary information from N input lines to a maximum of 2N unique output lines.

📖 Definition

A Decoder is a combinational circuit that converts binary information from N coded inputs to a maximum of 2N unique outputs. Only one output is active at a time, corresponding to the binary value of the inputs.

Active level: Outputs can be active-HIGH (selected output = 1) or active-LOW (selected output = 0).

💡 Applications

  • Memory Decoding: Selecting specific memory chips/addresses
  • Display Drivers: 7-segment displays, LCDs
  • Instruction Decoding: CPUs decode opcodes into control signals
  • Address Demultiplexing: Routing address lines to devices
  • Demultiplexer Implementation: Decoder + enable acts as DEMUX
  • Boolean Function Implementation: Any function via OR-gate of minterms

Common ICs

  • 74138: 3-to-8 Decoder
  • 74139: Dual 2-to-4 Decoder
  • 74154: 4-to-16 Decoder
  • 7447 / 7448: BCD-to-7-Segment Decoder
