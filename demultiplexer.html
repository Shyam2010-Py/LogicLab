🔀 Demultiplexer (DEMUX)

Routes a single input to one of many outputs based on select lines.

📖 Definition

A Demultiplexer (DEMUX) performs the reverse operation of a multiplexer. It takes a single input and routes it to one of many outputs, determined by select lines.

A 1-to-N DEMUX has 1 input, N outputs, and log₂(N) select lines.

⚙️ Working Principle

Only one output line is active at a time. The selected output receives the input value; all others are 0.

💡 Applications

  • Communication Systems: TDM demultiplexing — separating combined signals
  • Memory Decoding: Selecting specific memory chips or registers
  • Serial-to-Parallel Conversion: Distributing serial data to parallel outputs
  • Display Drivers: Routing data to specific display segments
  • Waveform Generators: Controlling signal routing
